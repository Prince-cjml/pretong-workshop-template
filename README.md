# Pretong Workshop Assignment

This is your personal assignment repository.

Do not clone the course upstream directly.

## Start the workshop

Run:

```bash
bash bootstrap.sh
```

The script will:

1. add the read-only course upstream;
2. fetch the versioned public release;
3. create the `submission` branch;
4. push `submission` to your personal repository.

After bootstrap:

```bash
git branch --show-current
```

Expected:

```text
submission
```

Continue by following the README now present on the `submission` branch.

Do not reset, rebase, delete, or force-push `submission`.
